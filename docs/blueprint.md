# **App Name**: Personal Investment Sage

## Core Features:

- User Authentication: Implement secure login and registration using Firebase Authentication, supporting email/password, Google, and Apple sign-in methods.
- Investment Management: Structure investment data in Firestore, including stocks, real estate funds (FIIs), fixed income, cryptocurrencies, ETFs, pensions, and cash. Each asset should store name, type, purchase date, purchase price, quantity, brokerage, and optional notes. The system calculates the average price, valuation, overall profitability and per asset and portfolio allocation by category.
- AI-Powered Investment Analysis: Provide a chatbot that analyzes the user's portfolio, identifies risk profiles based on assets and behavior (conservative, moderate, or aggressive), and generates personalized investment recommendations.
- Smart Alerts: The AI tool issues intelligent alerts for high volatility, excessive concentration, high-risk exposure, investment opportunities matching the user's profile, and maturity reminders.
- Minimalist UI Design: Create a clean, organized layout with light typography, ample spacing, simple, elegant charts, and intuitive navigation.
- Alert Display: Alerts should be clearly displayed.
- Snapshot Management: Periodically capture the state of the user's investments for historical tracking.

## Style Guidelines:

- Primary color: Soft green (#A7D1AB), evoking financial health and growth.
- Background color: Light green (#E5EBE3), desaturated to provide a clean, calming backdrop.
- Accent color: Muted blue (#77B5FE), for interactive elements, providing a trustworthy feel.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look suitable for both headlines and body text.
- Use simple, line-based icons to represent different investment types and actions, ensuring clarity and elegance.
- Employ a grid-based layout with ample white space to maintain a clean and organized presentation.
- Use subtle animations for loading states and transitions to enhance the user experience without being distracting.
